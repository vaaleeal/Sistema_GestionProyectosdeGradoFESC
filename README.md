# Portafolio UML Dinámicos  
## Sistema de Gestión de Proyectos de Grado – FESC

Este repositorio contiene el portafolio completo de diagramas UML dinámicos desarrollados para modelar el **Sistema de Gestión de Proyectos de Grado de la FESC**, como parte del taller de Análisis y Diseño de Sistemas.

El proyecto documenta el comportamiento integral del sistema, desde la creación de propuestas estudiantiles hasta la sustentación y archivo del documento final.

---

##  Contenido del Portafolio

### ✔ Diagramas UML Dinámicos
- **Casos de Uso:** Interacción entre actores (estudiantes, tutores, jurados, coordinación) y el sistema.
- **Especificaciones de Casos de Uso Críticos:** Detalle del caso “Sustentar Proyecto”.
- **Diagramas de Secuencia:** Flujo de mensajes durante la sustentación.
- **Diagramas de Comunicación:** Vista estructural equivalente a la secuencia.
- **Diagrama de Estados:** Ciclo completo del proyecto de grado.

---

## Objetivo del Sistema

Modelar un sistema institucional que permita gestionar de forma eficiente y controlada:

- Registro de propuestas y anteproyectos  
- Revisión y validación del tutor  
- Asignación de jurados  
- Programación de sustentaciones  
- Calificación y consolidación de notas  
- Archivo final del proyecto  

---

##  Herramientas Utilizadas

- **UML 2.0**
- **draw.io / diagrams.net**
- **Mermaid**
- **PlantUML (versión inicial)**
- **PDF para documentación final**
